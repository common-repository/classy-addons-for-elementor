== Description ==

Classy Addons for Elementor is an Ultimate essential addon for [Elementor](https://wordpress.org/plugins/elementor/) Page Builder with 10+ of the most popular and essential elements that you need for everyday use in the website page building.

Development of the [Classy Addons](https://classyaddons.com)  for Elementor has the worlds best practices in code standard and meets proper validation using the latest CSS, HTML , JavaScript and PHP 8.x technology to bring you a professional addon for the Elementor Page Builder Plugin that is WordPress 5.8.x ready and compliant.

Classy Addons for Elementor includes the most commonly used elements (called widgets) that help you easily manage your website content by simply using the drag and drop ability. There is absolutely no programming knowledge required, seriously!

== Installation ==

This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `classy-addons-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the elementor editor.
